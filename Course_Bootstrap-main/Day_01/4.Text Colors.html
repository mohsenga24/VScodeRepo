Contextual Colors

Use the contextual classes to provide "meaning through colors":

class:text-muted. This text is muted.

class:text-primary. This text is important.

class:text-success. This text indicates success.

class:text-info. This text represents some information.

class:text-warning. This text represents a warning.

class:text-danger. This text represents danger.

class:text-secondary. Secondary text.

class:text-dark. This text is dark grey.

class:text-body. Default body color (often black).

class:text-light. This text is light grey (on white background).

class:text-white. This text is white (on white background).
